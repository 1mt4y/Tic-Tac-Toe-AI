# Tic Tac Toe AI

A simple TicTacToe game with unbeatable AI using the MiniMax algorithm with alpha-beta pruning

# How To Run

Clone the repo:
```
git clone https://github.com/1mt4y/Tic-Tac-Toe-AI.git
```
Run the app:
```
python3.9 main.py
```
